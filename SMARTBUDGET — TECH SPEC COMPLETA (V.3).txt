1. 🚀 Overview

SmartBudget è una piattaforma completa per la gestione delle finanze personali composta da:
	•	Web App (core)
	•	Template Excel avanzato (complementare)
	•	Kanban e workflow di sviluppo
	•	Template per Notion
	•	Automazioni presenti e future (AI, PSD2, app mobile)

🎯 Obiettivi principali
	•	Gestione budget mensile/annuale
	•	Monitoraggio entrate, spese, risparmi, debiti
	•	Dashboard con grafici avanzati
	•	Calendario scadenze fatture/abbonamenti
	•	Analisi automatica (v2–v3) basata su AI

👤 Target
	•	Privati
	•	Coppie
	•	Famiglie
	•	Lavoratori autonomi con esigenze basiche

12. 📥 Guida passo passo — Import in Notion

1️⃣ Crea una pagina vuota
	•	Premi + Add Page
	•	Scegli Empty with Icon

2️⃣ Importa il CSV
	•	Dentro la pagina → Import
	•	Seleziona CSV
	•	Carica:
➡ SmartBudget_Kanban_Notion.csv

3️⃣ Cambia i tipi di colonne
	•	Priority → Select
	•	Complexity → Select
	•	Story Points → Number
	•	Status → Select (To Do, In Progress, Review, Done)

4️⃣ Crea la vista Kanban
	•	In alto → Add a View
	•	Scegli Board
	•	Raggruppa per Status

5️⃣ Aggiungi icone e colori
	•	Clicca su ogni label → imposta colore ed emoji come da label set ufficiale.

6️⃣ Aggiungi altre viste consigliate
	•	Priorità
	•	Tasks critiche
	•	Backend Only
	•	DevOps
	•	Design Tasks

7️⃣ Collega il Template Notion

Nella pagina principale → /linked database → scegli il database importato.
